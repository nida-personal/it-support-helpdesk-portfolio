# Helpdesk Scenario Simulations

These are independently built IT support practice scenarios created to demonstrate real-world remote helpdesk skills aligned with WFH IT support roles in Canada.

Each scenario includes a realistic support ticket, troubleshooting steps, resolution notes, and user-facing communication, mirroring actual Tier 1/2 helpdesk workflows.

| # | Scenario | Key Skills |
|---|----------|------------|
| 01 | Remote VPN Failure | VPN, Network, Remote Troubleshooting |
| 02 | MFA Lockout Remote User | Azure AD, M365, Identity Verification |
| 03 | New Hire Remote Onboarding | AD Provisioning, M365, VPN Setup |
| 04 | Slow Performance Home Office | Windows, Remote Diagnosis, Performance |

These are simulated scenarios created independently for portfolio purposes.
They reflect real support workflows based on enterprise IT experience.
