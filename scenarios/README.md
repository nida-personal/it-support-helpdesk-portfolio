## Overview

These are independently built IT support practice scenarios created to demonstrate
real-world remote helpdesk skills aligned with WFH IT support roles in Canada.

Each scenario includes a realistic support ticket, troubleshooting steps, resolution
notes, and user-facing communication, mirroring actual Tier 1/2 helpdesk workflows.

These scenarios are simulated and created independently for portfolio purposes.
They reflect real support workflows based on 3+ years of enterprise IT experience.

## Each Scenario Contains

- ticket.md — realistic support ticket with user details and issue description
- troubleshooting-steps.md — step by step diagnosis and actions taken
- resolution-notes.md — resolution summary, root cause, and knowledge base article
- user-communication.md — all emails and communication sent to the user

## Skills Demonstrated Across All Scenarios

- Remote troubleshooting without physical access to device
- VPN diagnosis and client version management
- Azure AD and MFA administration
- Active Directory user provisioning and account setup
- M365 licensing, Teams, Outlook, SharePoint and OneDrive support
- Remote new hire onboarding end to end
- Device performance diagnosis via Teams screen share
- Knowledge base article creation
- Professional user communication throughout ticket lifecycle
- Proactive follow-up and preventive recommendations
